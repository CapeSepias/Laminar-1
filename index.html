Simple

Approachable

Small library, no magic, no macros, almost no implicits.

Pragmatic

Easy concepts: no category theory, no typed effects, no backpressure, etc.

Native Scala.js

No dependencies on Javascript libraries or their concepts. No virtual DOM.

Expressive

Power of Scala

Use Scala FP and/or OOP techniques instead of having to learn ad-hoc library features.

Observables

First-class streams and state variables in one integrated system ‚Äì Airstream.

Many Building Blocks

Everything is abstractable and composable ‚Äì elements, attributes, attribute-value pairs, events, state, etc.

Safe

No FRP Glitches

Exposing inconsistent state is considered normal in most popular streaming libraries. But not in our Airstream.

Memory Management

Unique subscription ownership mechanism provides fully automatic memory safety without boilerplate.

Precise DOM Types

Proper types for DOM nodes and property / attribute values without macros or other complexity.

Easy Integrations

Web Components

Use Material UI or other Web Components just like any other JS dependencies.

React

Render Laminar elements inside React apps or the other way in just a few lines of code.

Anything Else

Laminar's direct API makes integrations with other DOM / streaming / etc. libraries a breeze.
